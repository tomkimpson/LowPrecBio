# Results Data

This directory contains experiment output data generated by the scripts in `scripts/`.
Data files (`.jld2`, `.csv`) are **not tracked by git** due to their size (~476 MB total)
and are stored locally only. Contact the authors if you need the raw data.

## Regenerating Results

Run the experiment scripts to regenerate all data:

```bash
julia --project=. scripts/run_all.jl
```

Or run individual models:

```bash
julia --project=. scripts/run_birth_death.jl
julia --project=. scripts/run_telegraph.jl
julia --project=. scripts/run_dimer.jl
julia --project=. scripts/run_repressilator.jl
julia --project=. scripts/run_schlogl.jl
```

## Directory Layout

- `birth_death/`, `telegraph/`, `dimer/`, `repressilator/` - Core experiment results (precision sweeps, 50k replicas)
- `schlogl/` - Schlogl bistable model (10k replicas; FP64/FP32/BF16 mixed + strict BF16+SR)
- `benchmarks/` - Wall-clock timing, memory, and throughput benchmarks (all 5 models + `summary.csv`)
- `publishability/` - Robustness experiments with multiple RNG seeds
  - `robustness/telegraph/` - 5 independent seed variations
  - `robustness/repressilator/` - 4+ independent seed variations
- `tmp_publishability/` - Smoke tests and exploratory runs (including Schlogl rescaled variants)
- `tmp_publishability_plan/`, `tmp_publishability_plan_r5/` - Additional robustness and parameter sweep runs

**Note:** Schlogl uses 10k replicas (not 50k) due to high per-trajectory event rates. FP16 modes excluded (cubic propensity overflow); strict BF16 RTN excluded (population runaway).

## File Types

- `ensemble_data.jld2` - Full ensemble trajectories (largest files, up to ~90 MB each)
- `metadata.jld2` - Run configuration and model parameters
- `extra_metrics.jld2` - Additional computed statistics (dwell times, conservation laws, etc.)
- `summary_stats.csv` - Per-mode summary statistics (mean, std, Wasserstein, KS tests)
- `benchmarks.csv` / `benchmarks.jld2` - Timing and memory measurements
